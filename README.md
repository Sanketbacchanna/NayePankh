# NayePankh Foundation Tech Internship Portal

The **NayePankh Foundation Tech Internship Portal** is a responsive web application designed to simplify the internship onboarding process. Built using **HTML5, CSS3, and JavaScript**, the portal provides an engaging user experience through interactive onboarding tasks, progress tracking, assignment unlocking, and dynamic development track selection.

---

## 🚀 Features

- Responsive and modern UI
- Internship onboarding checklist
- Progress tracking system
- Assignment unlocking mechanism
- Frontend, Backend, and Full Stack track selection
- Dark/Light mode toggle
- Interactive FAQ section
- Contact support form
- Smooth animations and confetti effects
- Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

---

## 🎯 Purpose

This project demonstrates modern frontend web development concepts, including:

- Responsive Web Design
- DOM Manipulation
- Event Handling
- Dynamic Content Rendering
- Interactive User Interfaces
- Theme Switching
- Progress Tracking

It serves as a lightweight internship portal without requiring any backend or database.
